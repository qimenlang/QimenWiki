手柄、手势事件处理逻辑区别

​	在多数事件的处理逻辑上保持一致

|               | 手柄              | 手势                                      |
| ------------- | ----------------- | ----------------------------------------- |
| scroll事件    | 支持              | 不支持，scroll 事件只有摇杆和滑动触屏产生 |
| translate事件 | 阈值判断位移+旋转 | 阈值只判断位移                            |
|               |                   |                                           |
